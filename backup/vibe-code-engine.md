# Project: Vibe Code Engine

**Category:** Coding / Automation / AI
**Year:** 2026
**Tech Stack:** Python, OpenClaw, SQLite, Shell

## Overview
The Vibe Code Engine is an autonomous market intelligence suite designed to automate job discovery and lead nurturing. It features a multi-agent orchestrator that handles scraping, semantic matching, and personalized outreach.

## Key Features
- **Multi-Agent Design:** Orchestrates specialized agents for different parts of the recruitment funnel.
- **Semantic Matching:** Uses local vector search to align project experience with job requirements.
- **Automated CRM:** Syncs found leads to Notion for tracking.
